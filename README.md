MyProfile Web merupakan aplikasi website profil pribadi berbasis framework Laravel yang dirancang untuk menampilkan informasi portofolio, sertifikat, keterampilan, dan profil secara dinamis. 

Website ini dilengkapi dengan sistem admin panel yang memungkinkan pengelolaan konten secara terstruktur, sehingga data dapat diperbarui tanpa perlu mengubah kode secara langsung.

Proyek ini dikembangkan sebagai bagian dari pengembangan portofolio pribadi sekaligus latihan implementasi framework Laravel dalam pengembangan aplikasi web berbasis MVC.
